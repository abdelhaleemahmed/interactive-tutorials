# Linux Administration Tutorial

**Interactive Tutorial** - Master Linux Command Line Administration

## Overview

A comprehensive, interactive tutorial for learning Linux system administration from fundamentals to advanced techniques. This tutorial combines theory, practical demonstrations, and hands-on exercises using an interactive terminal simulator.

## Tutorial Structure

### Core Topics

1. **Linux Fundamentals**
   - File system navigation
   - Directory structure understanding
   - User and permission basics
   - Introduction to the command line

2. **Command Line Mastery**
   - Essential commands and utilities
   - File manipulation and management
   - Text processing and searching
   - Command chaining and piping

3. **System Administration**
   - User account management
   - Group and permission management
   - Process management and monitoring
   - System information and logs

4. **Advanced Topics**
   - Shell scripting basics
   - Networking commands
   - Package management
   - System configuration and optimization

## Features

✅ **Interactive Terminal Simulator** - Run commands in the browser
✅ **Dark Mode Interface** - Comfortable for extended learning
✅ **Real Command Examples** - Copy and paste ready examples
✅ **Progressive Learning** - Start with basics, advance to complex tasks
✅ **Hands-on Exercises** - Practice what you learn immediately
✅ **Command Reference** - Quick lookup for common commands

## How to Use

1. **Open `tutorial.html`** in your web browser
2. **Start with the introduction** to understand core concepts
3. **Use the interactive terminal** to practice commands
4. **Follow the guided exercises** for each topic
5. **Refer to command reference** when needed

## Key Command Categories

### File Operations
```bash
ls, cat, cp, mv, rm, mkdir, rmdir, touch, find
```

### Text Processing
```bash
grep, sed, awk, sort, uniq, wc, cut, paste
```

### User Management
```bash
useradd, userdel, passwd, usermod, su, sudo
```

### Permissions
```bash
chmod, chown, chgrp, umask
```

### System Monitoring
```bash
ps, top, htop, df, du, free, uptime, iostat
```

### Package Management
```bash
apt, yum, dnf, pacman (depending on distro)
```

## Keyboard Shortcuts

| Shortcut | Action |
|----------|--------|
| `Ctrl+C` | Interrupt current command |
| `Ctrl+D` | Exit terminal |
| `Ctrl+L` | Clear screen |
| `↑/↓` | Command history |
| `Tab` | Auto-complete |

## Tips for Learning

1. **Practice regularly** - Linux skills develop through repetition
2. **Understand the "why"** - Don't just memorize commands
3. **Experiment safely** - Use the simulator for exploration
4. **Reference materials** - Keep common commands handy
5. **Read man pages** - Learn detailed command syntax and options
